  • Main Page

    • 메인페이지는 “세잎클로버”의 컨셉에 맞춘 그린 컬러와 프로필 사진 등으로 저의 아이덴티티를 표현하였습니다. 전체적인 분위기에 맞춰 색상과 타이포 그리고 레이아웃을 고려해서 통일성을 부여했습니다.

  • Branding Page

    • 브랜딩 페이지는 퍼스널 웹사이트의 기획과정이 담긴 페이지입니다. 저의 아이덴티티를 담은 로고의 제작 및 특성에 대해 설명하였습니다.

  • About Page

    • 어바웃 페이지는 디자이너로서 가진 역량과 목표를 보여주는 [나]를 소개해주는 페이지입니다. 개인 프로필 사진과 컬러 등 다양한 시각적인 요소를 이용해 효과적인 정보를 전달하기위해 노력하였습니다.

  • Portpolio Page

    • 포트폴리오 페이지는 각각의 작품들을 사용자가 보기 편리하게 작품의 특성을 고려하여 이미지와 색상을 선정하였습니다. 버튼을 이용한 슬라이드 형태로 제작하였습니다. 옅은 그림자를 주어 심미성과 차별성을 강화했습니다.

  • Experience Page

    • 경험 페이지에서는 디자이너로서 성장하기 전까지의 경험들을 배움, 디자인, 여행 등의 키워드로 정리하였습니다. 이해도를 높이기 위해 실제 경험했던 순간의 사진들을 첨부하여 표현하였습니다.

  • Publishing Page

    • Visul Studio Code를 이용해 Html & CSS를 기반한 개인 웹사이트를 제작하였습니다. 메인과 서브페이지에서는 Java Script를 통해 간단한 동적 효과를 구현하였습니다.

  • Epilogue

    • 나 자신을 효과적으로 표현하는 방법을 배우다.

      개인사이트를 제작 하는 과정에서 가장 기억에 남았던 부분을 내 스스로를 분석하고 이를 디자인으로 표현하는 것이었습니다. ‘나’라는 사람을 파악하고 브랜드화하기까지 어려웠 던 부분도 있었지만 어떠한 디자이너가 되고 싶은지에 대해 다시 생각해보게 되는 계기가 되었습니다. 제작 과정에 공을 들이고 기획의 컨쉡과 목표를 뚜렷하게 잡고 타깃층에게 어떻게 가장 편리하고 효율적으로 전달 할지 고민하는 시간을 가졌습니다.
